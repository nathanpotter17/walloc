LAYER_W: SUBMODULE_001: WALLOC

WebAssembly Memory Allocator

This page loads the WALLOC library from LAYER W (BSD3) for cross platform memory management.
Memory chunks are 64KiB, the max size of a WASM Page, however, the allocator can accept an arbitrary number of bytes above the minimum of 1 page. Total maximum allocation size is 65,535 pages.
(2^16 pages is 2^16 * 64KiB = ~4GiB)
